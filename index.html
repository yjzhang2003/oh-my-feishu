oh-my-feishu

在飞书里使用 Claude Code 的最佳方式

把飞书消息、Claude Code、本地项目目录和 lark-cli skills 连接起来，让 Claude Code 可以在飞书里被发起、继续、协作和回传结果。

Quick start

npm install -g oh-my-feishu
oh-my-feishu

Getting started

如何开始

README 里的安装和启动路径整理成四步，右滑到这一屏就能按顺序演示。

01

准备环境

确认终端可以运行 claude --version 和 lark-cli --version。Claude Code 负责对话与执行，lark-cli 负责飞书能力调用。

02

全局安装

npm install -g oh-my-feishu

03

启动配置

运行 oh-my-feishu，跟随交互式 CLI 完成检查、扫码授权和后台服务启动。

04

进入飞书

在飞书里发送 /menu，开始管理直接对话、目录会话、历史会话和自动化服务。

npm install -g oh-my-feishu oh-my-feishu

What you can do

你可以做什么

用菜单管理工作上下文 通过 /menu 切换直接对话、目录会话和历史会话。 在飞书里和 Claude Code 对话 不用切回终端，直接在飞书中讨论代码、查看回复和继续执行任务。 继续一段 Claude Code 会话 恢复电脑上的历史 session，或者绑定任意本地目录继续工作。 使用 Web Monitor 自动化服务 注册 traceback 地址后，后台轮询错误变化并触发 AI 分析或修复。 让 Claude Code 使用飞书能力 通过 lark-cli skills 操作文档、消息、日历、表格、任务和云盘。 只安装飞书能力插件 把 oh-my-feishu 作为 Claude plugin 安装到现有项目。
